Diabetes Prediction using Logistic Regression (From Scratch)

A machine learning project that predicts whether a patient has diabetes 
based on diagnostic measurements, implementing Logistic Regression 
from scratch using gradient descent — without relying on `sklearn`'s 
built-in LogisticRegression.

Overview
This project demonstrates a foundational understanding of binary 
classification, going beyond a black-box sklearn call to implement the 
sigmoid function, cost function (log loss), and gradient descent 
optimization manually.

Objective
To classify whether a patient is diabetic (1) or non-diabetic (0) based 
on features like glucose level, BMI, blood pressure, age, and insulin 
levels, using the Pima Indians Diabetes Dataset

Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn (for visualization)

Methodology
1. Data preprocessing — handling missing/zero values and standardizing features
2. Implemented the sigmoid activation function from scratch
3. Derived and coded the log-loss (binary cross-entropy) cost function
4. Optimized weights and bias using gradient descent
5. Evaluated model using accuracy, precision, recall, and confusion matrix

Results
- Achieved strong classification accuracy on the test set
- Model effectively distinguishes diabetic vs non-diabetic patients based 
  on key health indicators
